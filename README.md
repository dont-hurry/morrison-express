如果後端是以 RESTful API 的方式實作，前端 CRUD 操作通常會用到以下 API：

### Create

```txt

```

```txt
GET /api/items

```

```txt
PATCH /api/items/itemId
```

```txt
DELETE  /api/items/itemId
```
